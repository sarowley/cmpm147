Experiment 5 - Monkey Madness Monday

Description

This project is a Voronoi diagram based genetic algorith that is used to recreate images using general shapes and colors.

Technical

The algorithm starts by choosing a background color and a few points to base the Voronoi diagram around. It then goes through and for every 4th point will determine which Voronoi point it is closest to and chooses its respective color. It will then move the points around checking its fitness against an evaluation function to see how similar it is to the target image and mutating off of the best fit.
