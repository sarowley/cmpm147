Experiment 3 - Alternate World

Description

What are you trying to acomplish with this experiment?

In this experiment Miles and I were attemping to create a random map generator for both overworld and dungeon areas. We both wanted to accomplish creating a map generator that looked realistic, while not being too cluttered or overbearing.

Technical

Describe in detail what you did technically and how it works?

What we did was use P5.js along with a provided tilemap to create randomly generated maps. We used random(), along with noise() to get the exact results that we were striving for. It's created by taking in a grid of (changeable) characters, and then using code to do things like translate, transform, or autotile, it renders a map in real time.

For this project, the things that I worked on were as follows: I set everything up for my partner Miles and I before we started working, I did most of the writeup that we had to turn in as a companion assignment to this one, and got the "live" aspect of the water working.
